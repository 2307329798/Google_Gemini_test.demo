# Google_Gemini_test.demo

**一个简单的用于测试 Google Gemini 模型的案例库。**

## 📂 项目目录与功能介绍

本项目包含以下测试文件：

### 1. 🎄 动态粒子圣诞树 (`chrismastree.html`)
这是一个 3D 粒子圣诞树程序。
- **核心功能**：通过 AI 视觉算法检测手势。
- **交互方式**：识别手部的**开合**动作。当你张开或握拳时，可以动态控制圣诞树粒子的聚散状态。
- **彩蛋**：双手打开，你将能看到闪烁的节日祝福“圣诞快乐”，可以在对应位置修改你的祝福语。

### 2. 🖐️ AI 粒子交互 (`index.html`)
这是圣诞树粒子动态控制的基础版本。
- **功能描述**：构建了 3D 粒子渲染的底层逻辑。
- **交互特点**：集成了 AI 手势控制功能，允许用户通过手势与 3D 粒子进行互动。同时，页面包含了一个简单的 UI 交互界面，用于调整参数或选择模型。

### 3. 🎮 闯关小游戏 (`game.html`)
一个轻量级的网页版闯关游戏，宝剑小人击杀NPC进行升级最终击败BOSS获得游戏胜利。
- **玩法**：简单的逻辑关卡设计，用于测试代码生成的逻辑连贯性和游戏循环机制，还不够完善。

### 4. 📚 单词记忆练习 (`word_quiz.html`)
专注于单词记忆功能的简洁界面，部分功能仍待完善。
- **功能**：提供一个循环背单词的交互界面，帮助用户进行词汇记忆训练。

## 🚀 如何运行

1.  在浏览器中访问对应的文件如chrismastree（`(https://2307329798.github.io/Google_Gemini_test.demo/chrismastree)`）。
2.  允许浏览器访问摄像头权限以体验手势控制功能。
